Input Translator Assistant ("the Extension") is operated by the developer of this project. This policy explains what data is processed when you use the Extension.

## 1. Data we process

The Extension processes the following data only when needed for translation:

- Text you select or type for translation.
- Target language setting you choose.
- Basic request diagnostics (for example request ID, trace ID, and error stage) for troubleshooting.

## 2. How data is used

We use the above data only to provide core functionality:

- Generate translation options (natural, formal, concise).
- Return translation results to your browser.
- Diagnose service failures and improve reliability.

## 3. Where data is sent

To provide translation, selected text is sent to:

- A developer-operated backend service (Cloudflare Workers).
- A third-party model/translation provider used by that backend (currently SiliconFlow API compatible endpoint).

## 4. Data storage and retention

- Translation text is processed for request handling.
- We do not intentionally sell user data.
- We do not use translation text for advertising.
- If logs are enabled for debugging, logs are limited to operational diagnostics and are not intended to store full user raw text.

## 5. Data sharing

We share data only with service providers required to deliver translation functionality (for example cloud hosting and model API providers).

## 6. Your choices

- You can stop data processing by disabling or uninstalling the Extension.
- You can avoid sending specific text by not triggering translation for that text.

## 7. Security

We take reasonable measures to protect transmitted data, including HTTPS transport and access controls where supported.

## 8. Changes to this policy

We may update this policy from time to time. Updated versions will be published at the same policy URL with a new "Last updated" date.
